# form2jsonschema(temporary)

> ## 一个简单的 [React](http://facebook.github.io/react/) 组件，利用Form表单的形式来构造[JSON schema](http://jsonschema.net/)，UI使用 [Ant Design](https://ant.design/index-cn)。

## Schema

- 最外层是一个唯一的object
  - 可以建立properties
  - 设置required
  - 设置title
  - 设置description
  - 设置definitions
- 可建立以下类型的property
  - string类型
    - 选择所属的对象
    - 设置key
    - 设置title
    - 设置description
    - 设置default
    - 选择ui
    - 设置format形式
  - object类型
    - 选择所属的对象
    - 添加properties
    - 设置key
    - 设置title
    - 设置description
    - 设置default
    - 选择ui
    - 设置required
  - number类型
    - 选择所属的对象
    - 设置title
    - 设置description
    - 设置default
    - 设置最大值
    - 设置最小值
    - 选择ui
    - 使用enum
      - 使用enum时可设置uniqueItems(成员是否唯一)
  - boolean类型
    - 选择所属的对象
    - 设置title
    - 设置description
    - 设置default
    - 选择ui
  - array类型
    - 选择所属的对象
    - 设置title
    - 设置description
    - 设置default
    - 设置固定成员（items为数组）
      - 可删除固定成员
    - 设置可选成员（items为对象）
      - 可以重置已设置的items
      - items可设置一下key
        - default
        - enum
          - 使用enum时可设置uniqueItems(成员是否唯一)
    - 设置items，一个对象，成员都按照对象内的描述来创建
